# Employee Salary Processing System

This is a simple Java application that calculates the salaries for different types of employees using Object-Oriented Programming (OOP) concepts.

## Features
* **Inheritance:** Uses a superclass (`Employee`) and a subclass (`Manager`) to differentiate salary structures.
* **Method Overloading/Overriding:** Calculates standard wages and adds bonuses for managers.
* **Array & String Operations:** Stores employee names in an array and formats them (capitalization, length checking, and substring searches).

## Technologies Used
* Java

## How to Run in Visual Studio Code
1. Ensure you have the Java Development Kit (JDK) installed.
2. Open the `SalaryProcessing.java` file in VS Code.
3. Click the **Run** button above the `main` method, or run `java SalaryProcessing.java` in the terminal.
